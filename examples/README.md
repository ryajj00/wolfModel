# wolfModel Examples

## basic_verify

Full lifecycle demo: load a signed `.wmdl` model file, verify it, and print the result.

Runs on any platform with wolfSSL installed: Raspberry Pi 4/5, Linux, macOS.

### Build

From the repository root:

```bash
make                    # build libwolfmodel.a
make examples           # build the example (output: build/wolfmodel_verify)
```

Or from this directory:

```bash
cd examples/basic_verify
make
```

### Usage

```bash
# Generate a test keypair and signed model
make keytools
make fixtures

# Verify the signed model
build/wolfmodel_verify tests/fixtures/dummy.wmdl tests/fixtures/pubkey.der
```

Expected output:

```
Model verified OK.
  Version: 1
  Payload: 4096 bytes
```

### Key format

Public key files are raw `Qx + Qy` (64 bytes for ECC-256). Generated by `wolfmodel_keygen`.
This is the same raw key format used by wolfBoot.
